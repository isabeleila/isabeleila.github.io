Barbershop Simulator, Spring 2024 Semester

Created a Java code that simulates the wait times in a barbershop by using a circular queue. The program reads through a list of people that are there for a haircut, checks if there is an open chair and if there is allows the next person to sit, but if not then they are added to the wait until the time for the person in the barberchair is up.
